# Windows-Capture-Zone

**Capture and share only a chosen area of your screen**

</div>

---

### What it is

- A resizable frame you place over any content.
- A companion window named `<<<CAPTURE ME>>>` that mirrors only the framed area.

### How to use

1. Run the app.
2. Move/resize the frame around what you want to share.
3. In your call/recording app, share the window named `<<<CAPTURE ME>>>`.

### Compatibility

Works with any app that supports sharing a specific window, including:

- Zoom
- Microsoft Teams
- Google Meet
- Discord
- Slack
- Telegram
- WhatsApp
- Webex
- and more…
